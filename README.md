# ArithmeticCode

Demonstração de **codificação aritmética** em C: modelo empírico a partir da mensagem, intervalo em `[0, 1)` em `long double`, codificação e decodificação reversíveis dentro dos limites de precisão.

## Compilar e executar

```bash
gcc -std=c11 -Wall -Wextra -o arithmetic_demo main.c utils.c arithmetic.c -lm
./arithmetic_demo
```
